# **모두의 파이썬**  

## **개요**  
이 레포지토리는 **2022년 1월 28일**부터 **2월 14일**까지 **모두의 파이썬** 교재를 기반으로 파이썬의 기초 개념을 정리한 것입니다. 파이썬 설치부터 연산, 반복문, 함수, 그래픽 응용까지 다양한 기초 문법과 실습 프로젝트를 학습하였습니다.

## **주요 특징**  
- 파이썬 설치 및 기초 프로그램 작성  
- 연산, 변수, 반복문, 함수 학습  
- 거북이 그래픽 및 다양한 프로젝트 진행

## **기술 스택**  
- **Python**
